# Tikrit

Tiny survival-horror [roguelike](https://en.wikipedia.org/wiki/Roguelike) written in [Lua](https://www.lua.org/) using Love2D.

## Usage

```console
$ git clone https://github.com/gongahkia/tikrit
$ ./install.sh
$ ./build.sh
$ make
$ make test # optionally
```
